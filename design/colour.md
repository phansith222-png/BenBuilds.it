# Colour Palette

All tokens are defined in `tailwind.config.ts`. Always use these — never hardcode hex values in components.

| Token | Hex | Usage |
|---|---|---|
| `ink` | `#1A1A1A` | Dark section backgrounds (Hero, Origin, Works, Capabilities, Contact) |
| `paper` | `#F5F5F5` | Light section background (Trilingual / Languages) |
| `orange-accent` | `#FD6F00` | Accent highlights, CTAs, hover states |
| `text-black-main` | `#FFFFFF` | Primary text on dark (ink) backgrounds |
| `text-black-sub` | `#A3A3A3` | Secondary/muted text on dark backgrounds |
| `text-gray-main` | `#1A1A1A` | Primary text on light (paper) backgrounds |
| `text-gray-sub` | `#4A4A4A` | Secondary text on light backgrounds |

> `gray-panel` (#D9D9D9) and `black-panel` (#000000) are kept in the config for now but are no longer used — remove them once confirmed clean.
