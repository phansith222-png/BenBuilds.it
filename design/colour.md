# Colour Palette

All tokens are defined in `tailwind.config.ts`. Always use these — never hardcode hex values in components.

| Token | Hex | Usage |
|---|---|---|
| `gray-panel` | `#D9D9D9` | Left panel background |
| `black-panel` | `#000000` | Right panel background |
| `text-gray-main` | `#1A1A1A` | Primary text on gray |
| `text-gray-sub` | `#4A4A4A` | Subtext on gray |
| `text-black-main` | `#FFFFFF` | Primary text on black |
| `text-black-sub` | `#A3A3A3` | Subtext on black |
| `orange-accent` | `#FD6F00` | Accent highlights |
